Javascripts is synchronus single-threaded High-level interpreted programming language. Everything occurs inside javascript is execution context. There are Two execution context components which are memory component and thread of execution. Memory component stores all the variables and functions as key-value pairs. And Thread of executes all codes by one line at a time.
